== Description ==

Maximus Movie Video is a lightweight, movie video WordPress theme perfect for video creators, movie bloggers, and content publishers. Designed with performance and SEO in mind, Maximus Movie Video features clean, accessible code, integrated schema markup, and fast page loads. The theme supports multiple post formats including video, gallery, and standard posts, along with infinite scroll and related posts to enhance user engagement. With full support for the WordPress Customizer, Maximus Movie Video lets you easily personalize layouts, colors, and typography. It’s translation-ready, responsive, and built to work seamlessly with Gutenberg and popular plugins. Whether you’re building a film review site, a video showcase, or a personal blog, Maximus Movie Video provides a clean, modern design and all the tools you need to grow your audience. Theme Demo here https://demo.vinethemes.com/maximus-movie-video/.

== Installation ==

1. In your admin panel, go to Appearance -> Themes.
2. Click Add New.
3. Click Upload.
4. Click Choose File and select the theme zip file you downloaded from WordPress.org.
5. Click Install Now.
6. Click Add New, then click Upload, then click Choose File.
7. After WordPress installs the theme, click Activate.
8. You've successfully installed your new theme!
